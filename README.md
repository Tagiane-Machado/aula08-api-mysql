# Aula 08 - API MySQL

Projeto desenvolvido para estudos de integração entre Node.js e MySQL.

## Tecnologias utilizadas

* Node.js
* Express
* MySQL
* Git e GitHub

## Instalação

Clone o repositório:

```bash
git clone https://github.com/Tagiane-Machado/aula08-api-mysql.git
```

Instale as dependências:

```bash
npm install
```

## Configuração

Crie um arquivo `.env` com as credenciais do banco de dados.

Exemplo:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=sua_senha
DB_NAME=nome_do_banco
```

## Executando o projeto

```bash
npm start
```

ou

```bash
npm run dev
```

## Objetivo

Praticar operações de API conectadas a um banco de dados MySQL, utilizando boas práticas de versionamento com Git e GitHub.
